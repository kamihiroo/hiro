Progateの特徴

1、初心者にやさしい

プログラミング初心者にとってまず最初に壁になるのは 環境の構築です。
ですがProgateなら初めから環境が用意されているのでその手間なくプログラミングを学び始めることができます。
初心者の方でも、すぐにプログラミングに触れることができますよ。

2、丁寧でわかりやすい学習方式

Progateの学習方法はスライドショーを使って学んでいきます。
丁寧でわかりやすいスライド説明のあとに、実際に自らの手を動かしてプログラムを書いていきます。
もしも分からなくなった場合でもスライドの確認、ヒント、答えを見ることができるのでエラーにつまずくことなく学習を進めていけます。

対応言語一覧

HTML & CSS

ウェブページの作成に使用される言語です。HTMLとCSSを組み合わせることで、静的なページを作り上げることができます。

jQuery

素敵な動きを手軽に実装できるJavaScriptライブラリです。 アニメーション効果をつけたり、Ajax（エイジャックス）を使って外部ファイルを読み込んだりと色々なことができます。

PHP

HTMLだけではページの内容を変えることはできません。PHPはHTMLにプログラムを埋め込み、それを可能にします。
